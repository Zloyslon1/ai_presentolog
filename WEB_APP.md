# Flask Web Application for Presentation Design System

**Simple web interface using Python Flask instead of Rails**

Since Rails requires WSL setup, we'll use Flask (Python) for the web interface. This integrates directly with the existing Python presentation design system.

## Features

- Web form to paste Google Slides URLs
- Template selection dropdown
- Real-time processing status
- Download/view designed presentations
- Processing history

## Quick Start

```bash
# Install Flask
pip install flask

# Run web server
python web_app.py

# Open browser
http://localhost:5000
```
